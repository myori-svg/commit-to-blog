# .claude/commands/review.md

현재 프로젝트 코드 전체를 검토하고 아래 항목 기준으로 리뷰해줘:

1. 아키텍처 — 컴포넌트 구조, 폴더 구조가 CLAUDE.md 기준에 맞는지
2. 코드 품질 — any 타입, 불필요한 중복, 매직 넘버 등
3. 스타일 일관성 — Tailwind 외 다른 방식 혼용 여부
4. 디자인 스펙 — docs/design-system.md 기준에서 벗어난 부분
5. 개선 제안 — 리팩토링이 필요한 부분

문제가 없으면 "이상 없음"으로 표시. 
수정이 필요한 경우 사용자 확인 후 진행할 것.
